Тобі потрібно розділити заданий текст на слова. Орієнтуйся при цьому на "_". Кожне слово, яке було після "_", а також перше слово у тексті, потрібно зробити з великої букви.
Мені потрібна допомога, щоб продовжити місію
Тобі буде корисно прочитати трохи інформації про рядковий метод: str.split().
Потрібна допомога!
Подивись, як працює метод str.split(): 
text = "Hello, world!"
text.split()  # По замовчуванню, str.split() повертає список слів у рядку,
              # використовуючи "sep" як пробіли
>> ['Hello,', 'world!']

text.split('o')  # Якщо ми введемо "sep" як літеру "o", ми побачимо наступне
>> ['Hell', ', w', 'rld!']
Мені потрібна маленька підказка.
Давай для рішення використаємо вираз генератор. Ось в чому ідея: вираз генератор тобі допоможе без лишніх змінних пройтись по списку розділених слів, та зробити їх з великої букви. Ось приклад: 
txt = "this_function_is_empty"
txt = (i.capitalize() for i in name.split('_'))
>>  ['This', 'Function', 'Is', 'Empty']  # Для того, щоб побачити результат,
                                         # Треба використати list() на вираз
Нічого не працює. SOS
Все, що тобі залишилось це з'єднати список слів назад у рядок. Використай str.join() 
''.join(i.capitalize() for i in name.split('_'))
>>  "ThisFunctionIsEmpty"
Останнє запитання. Як виглядатиме моє остаточне рішення? 
def to_camel_case(name: str) -> str:
    return ''.join(i.capitalize() for i in name.split('_'))
