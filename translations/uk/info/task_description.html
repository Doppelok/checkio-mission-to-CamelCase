Твоя місія полягає в тому, щоб перетворити назву функції (у вигляді рядка) з формату Python (наприклад, "my_function_name") у CamelCase ("MyFunctionName")

, де перша літера кожного слова знаходиться у верхньому регістрі, і всі слова об'єднані без будь-яких символів.

Вхідні дані: Назва функції (рядок).

Вихідні дані: Назва функції CamelCase (рядок).

Приклад: 

{{init_code_tmpl}}

Як це використовується: Застосовувати назви функцій у стилі, в якому вони приймаються певною мовою програмування (Python, JavaScript тощо).

Передумови::
0 < len(string) <= 100
Вхідні дані не містять жодних чисел.
